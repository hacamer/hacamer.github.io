扩展

在有条件的情况下 不推荐在AdGuard安装扩展 建议在浏览器内安裝

关闭AdGuard自动检测用户脚本

Windows：

设置=>扩展=>取消勾选自动检测用户脚本

Mac：

高级设置=>subscription.link.interception.userscript 调成false

设置完成

备注

如果你搭配了Clash等网络工具使用，请开启过滤本地连接

Windows：

设置=>常规=>高级设置=>过滤本地连接

Mac：

高级设置 => network.filtering.localnetwork 和 network.filtering.localhost 调成 true
